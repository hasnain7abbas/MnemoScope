# MnemoScope brand

The mark combines three ideas: an `M` for memory, a scope ring, and an amber
wave that behaves like both a thought signal and a timeline. The warm amber
identifies moments in time; cyan identifies navigation, focus, and discovery.

- `mnemoscope-mark.svg` is the compact navigation mark.
- `mnemoscope-icon.svg` is the square application icon source.
- `mnemoscope-wordmark.svg` is the horizontal lockup.
- `mnemoscope-logo.svg` is the centered presentation lockup.

Keep clear space around the mark equal to the width of its inner `M` stroke.
Do not recolor the wave independently of the amber brand token.
