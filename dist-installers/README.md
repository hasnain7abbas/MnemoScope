# MnemoScope 0.1.0 Windows builds

- `MnemoScope_0.1.0_x64-setup.exe` is the recommended NSIS installer.
- `MnemoScope_0.1.0_x64_en-US.msi` is the Windows Installer package.

Both artifacts were produced with:

```bash
npm run tauri build
```

Verify them with the SHA-256 values in `SHA256SUMS.txt`.
